# Examples - E-commerce

This folder contains reference projects for the e-commerce sector.

## Purpose

Projects stored here serve as design and implementation references for new e-commerce clients.

Includes: product stores, DTC brands, boutiques, online retail.

## When to Use

- Before starting a new e-commerce project, review existing examples for pattern reference.
- Use examples to calibrate dials (Variance, Motion, Density) for the sector.
- Do not copy content from examples. Use them for structural and design reference only.

## Sector Notes

- E-commerce projects prioritize product imagery and social proof above all else.
- Motion dial is typically medium (4-7). Product interaction drives engagement.
- Primary conversion: add to cart or direct purchase.
- Trust signals: reviews, return policy, secure payment badges.

## Adding a New Example

After completing an e-commerce project, the EscalaSites team may promote it to this folder as a reference.

Only completed, QA-approved projects should be added as examples.

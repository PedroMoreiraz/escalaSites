# Examples - Clinic

This folder contains reference projects for the clinic and healthcare sector.

## Purpose

Projects stored here serve as design and implementation references for new clinic clients.

Includes: medical clinics, dental practices, wellness centers, physiotherapy studios.

## When to Use

- Before starting a new clinic project, review existing examples for pattern reference.
- Use examples to calibrate dials (Variance, Motion, Density) for the sector.
- Do not copy content from examples. Use them for structural and design reference only.

## Sector Notes

- Clinic projects require high trust signals: credentials, certifications, team photos.
- Tone is authoritative and warm. Never casual.
- WCAG AA compliance is non-negotiable for healthcare.
- Booking/appointment CTAs are the primary conversion goal.

## Adding a New Example

After completing a clinic project, the EscalaSites team may promote it to this folder as a reference.

Only completed, QA-approved projects should be added as examples.

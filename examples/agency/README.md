# Examples - Agency

This folder contains reference projects for the agency sector.

## Purpose

Projects stored here serve as design and implementation references for new agency clients.

Includes: creative agencies, marketing agencies, design studios, consulting firms.

## When to Use

- Before starting a new agency project, review existing examples for pattern reference.
- Use examples to calibrate dials (Variance, Motion, Density) for the sector.
- Do not copy content from examples. Use them for structural and design reference only.

## Sector Notes

- Agency projects have the highest variance (8-10). The site is itself a portfolio piece.
- Motion dial is typically high (7-10). Motion demonstrates craft.
- Primary conversion: project inquiry or case study engagement.
- Portfolio work must be real and with client permission.
- The EscalaSites design system must not be visibly recognizable on agency sites.

## Adding a New Example

After completing an agency project, the EscalaSites team may promote it to this folder as a reference.

Only completed, QA-approved projects should be added as examples.

# Examples - Restaurant

This folder contains reference projects for the restaurant sector.

## Purpose

Projects stored here serve as design and implementation references for new restaurant clients.

Each reference project demonstrates how EscalaSites principles are applied to a specific restaurant context.

## When to Use

- Before starting a new restaurant project, review existing examples for pattern reference.
- Use examples to calibrate dials (Variance, Motion, Density) for the sector.
- Do not copy content from examples. Use them for structural and design reference only.

## Structure (when populated)

```
restaurant/
    [client-slug]/
        docs/
        site/
        assets/
```

## Adding a New Example

After completing a restaurant project, the EscalaSites team may promote it to this folder as a reference.

Only completed, QA-approved projects should be added as examples.

# Content Workflow

Goal:

Transform strategy into implementation content.

Inputs:

* brand-profile.md
* page-architecture.md
* content-strategy.md
* site-config.md

Steps:

1. Generate Data Layer.
2. Validate content consistency.
3. Validate CTA alignment.
4. Validate trust elements.
5. Validate SEO direction.

Outputs:

* data-layer.md

Validation:

* Hero supports conversion.
* Services reinforce positioning.
* FAQ reduces objections.
* CTA supports primary goal.

Do not proceed until all validations pass.

# Strategy Workflow

Goal:

Transform research into strategy.

Inputs:

* discovery-notes.md
* research-summary.md

Steps:

1. Generate Brand Profile.
2. Generate Page Architecture.
3. Generate Content Strategy.
4. Generate Site Config.

Outputs:

* brand-profile.md
* page-architecture.md
* content-strategy.md
* site-config.md

Validation:

* Positioning is clear.
* Audience is clear.
* Messaging is clear.
* Conversion path is clear.

Do not proceed until all validations pass.

# Quality Checklist

## Strategy

- [ ] Positioning is clear
- [ ] Audience is clear
- [ ] Offer is clear
- [ ] Conversion goal is clear

## Content

- [ ] No generic marketing language
- [ ] No AI clichés
- [ ] No vague claims
- [ ] Headlines are specific

## Design

- [ ] Mobile-first
- [ ] Proper spacing hierarchy
- [ ] Consistent typography
- [ ] Consistent CTA hierarchy

## Development

- [ ] No duplicated code
- [ ] Data-driven implementation
- [ ] Components reusable
- [ ] Type-safe implementation

## Final

- [ ] Site feels custom
- [ ] Site supports business goals
- [ ] Site follows EscalaSites principles
